# XBoard 插件仓库

本仓库为 XBoard 的插件集合，包含 Mala Theme Team 开发的多个实用插件，帮助扩展 XBoard 系统功能。

## 📦 插件列表

### AutoReply [免费]

智能客服自动回复插件

- 支持关键词自动回复
- 集成 OpenAI GPT 智能回复
- 支持转人工处理
- 支持 Telegram 通知

[查看详细文档](./AutoReply/README.md)

## 📖 使用说明

### 安装插件

1. 下载插件目录到 XBoard 的 `/plugins` 目录下
2. 在 XBoard 管理后台启用插件
3. 根据插件文档进行配置
